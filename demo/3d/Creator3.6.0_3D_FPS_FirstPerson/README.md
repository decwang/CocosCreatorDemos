### Support
Game operation keys:
- W = Forward
- A = Left
- S = Back
- D = Right
- shift + W = Accelerated Running
- C = Squat
- Mouse Left = Shoot
- Mouse Right = Aim
- Space = Jump
- Q = Elbow Strike
- R = Change Magazines
- Mouse Wheel = Switching Guns
